竞品分析

发表于 2019-04-16

Bernard Shen

竞品调查

在iPhone appstore上搜索关键词“文言文”和关键词“古文”，选取其中评论数大于200的搜索结果进行分析

搜索结果

品名 appstore评分（满分5分） 评论数
文言文大全 4.6 377
文言文字典 4.8 1120
百度汉语 4.8 44300
西窗烛 4.9 134000

特色分析

  1. 文言文大全
    • 优点
      • 与多本教材对接
      • 提供古诗文原文与译文（分离给出，非一句一译）
      • 提供文化背景常识普及
    • 缺点
      • 无加点字解释
      • 无主动推送学习任务
      • 无每日一句
      • 无查词功能
  2. 文言文字典
    • 优点
      • 文言文生字查询
      • 提供生词本
      • 有例句
    • 缺点
      • 无主动推送学习任务
      • 无每日一句
  3. 百度汉语
    • 优点
      • 每日一荐（推送成语及出处）
      • 拍照朗读（ai识别文本，给出拼音）
      • 字词听写（替代家长默写）
      • 诗词背诵
      • 趣味游戏
      • 生词本（成语）
    • 缺点
      • 无主动推送学习任务
  4. 西窗烛
    • 优点
      • 诗词检索
      • 名句卡片
      • 一日一赏（每天推荐一首诗词）
      • 诗歌朗诵
      • 诗词游戏
    • 缺点
      • 无主动推送学习
      • 无查词功能
      • 针对古诗

总结

综上可知，市面上的热门文言文、古文学习app大多停留在等待用户自己主动学习的阶段。我们的核心功能要放在实现像现今英文背词app一样的，用户在第一天制定学习计划，之后由app主动推送当天的学习任务。
对于用户较多的两款app：百度汉语和西窗烛，前者目标对象主要为小学生，目标是学习语文；后者目标对象主要为古诗文爱好者、主要为成年人，目标是增加古诗词赏析水平。可见我们app与这两款现有的app无论是目标用户还是目标上都不冲突。
